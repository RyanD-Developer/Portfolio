Tiny Spirits

Tiny spirits is a 3D Action/Adventure game. It was developed with free assets since Ashton and I arent great artists, it was made in unreal engine, for a project in my college course. I was in charge of player movement, Level 2's design, the cat bossfight, the sfx and the ui. This is a dream come true, being able to work on a passion project like this and to see peoples reactions to me and ashtons hard work, although the game is in an unpolished state im very proud of the game for only 3 months work and my first big 3d project in unreal (i say big because I made a small racing game with one level but it wasnt that interesting nor as ambitous).
